       - executeBuy(signal: ValidatedSignal): Promise<TradeResult>
       - executeSell(position: Position, reason: ExitReason): Promise<TradeResult>

    2. Implement SafeExecutor class:
       - Constructor takes config (for buyAmountSol)
       - executeBuy(signal):
         a. Generate a mock entry price. Since we don't have the price monitor wired yet, use a placeholder price approach: store a `getPrice` callback that will be set later when price monitor is ready. For now, use a small random price (e.g., 0.00001 SOL range) as fallback.
         b. Calculate tokensReceived = buyAmountSol / entryPrice
         c. Return TradeResult with signature = `SAFE_BUY_${Date.now()}_${randomHex(8)}`, mode = 'SAFE'
       - executeSell(position, reason):
         a. Use position.currentPrice as exit price
         b. Calculate solReceived = position.tokensHeld * exitPrice
         c. Return TradeResult with signature = `SAFE_SELL_${Date.now()}_${randomHex(8)}`, mode = 'SAFE', reason

       Helper: randomHex(bytes: number) returns crypto.randomBytes(bytes).toString('hex')

    3. Export function initTradeExecutor(config: Config): void
       - Create SafeExecutor instance
       - Subscribe to bus.on('signal:validated', async handler):

         c. If allowed: call executor.executeBuy(signal), emit bus.emit('trade:executed', result)
       - Log each trade execution at info level

    4. Export the SafeExecutor class separately so it can be used by position manager for sells.
       Also export a function to set the price getter: setPriceGetter(fn: (mint: string) => Promise<number>)
       This allows the price monitor (Plan 04) to inject real price lookup without circular deps.

<task type="auto">
  <name>Task 2: Position manager with persistence and exit checking</name>
  <files>src/pipeline/position-manager.ts</files>
  <action>
    Create src/pipeline/position-manager.ts:

    1. Export class PositionManager:
       - Private state:
         positions: Map<string, Position> (keyed by mint)
         tradedMints: Set<string> (all-time: active + closed history)
         history: Position[] (closed positions)

       - Constructor: takes config (maxPositions, buyAmountSol)

       - async init(): Load state from data/positions.json and data/history.json using readJson utility.
         Rebuild positions Map and tradedMints Set from loaded data.
         Log how many open positions and historical trades were loaded.

       - canOpenPosition(mint: string): { allowed: boolean; reason?: string }
         Check 1: positions.size >= config.maxPositions -> { allowed: false, reason: 'Max N concurrent positions' }
         Check 2: tradedMints.has(mint) -> { allowed: false, reason: 'Already traded this mint' }
         Otherwise: { allowed: true }

       - openPosition(result: TradeResult, copiedFrom: string): Position
         a. Generate randomized exit params:
            takeProfitPct: random between 100 and 200 (integer)
            stopLossPct: random between -50 and -30 (integer, negative)
            timeLimitMs: random between 15 and 45 minutes (in ms)
         b. Create Position object with all fields
         c. Add to positions Map and tradedMints Set
         d. Persist state (call this.persist())
         e. Emit bus.emit('position:opened', position)
         f. Log: "Position opened: {mint short} | TP: {tp}% | SL: {sl}% | Time: {min}min"
         Return the position

       - checkExits(prices: Map<string, number>): void
         For each open position:
         a. Get current price from prices map. If not available, skip.
         b. Update position.currentPrice and position.pnlPct = ((currentPrice - entryPrice) / entryPrice) * 100
         c. Check exit conditions in PRIORITY ORDER:
            1. Stop-loss: pnlPct <= exitParams.stopLossPct
            2. Take-profit: pnlPct >= exitParams.takeProfitPct
            3. Time-limit: Date.now() - openedAt >= exitParams.timeLimitMs
         d. If any condition met, call this.closePosition(mint, reason)
         e. Otherwise, emit bus.emit('position:updated', position)

       - checkTimeLimitExits(): void
         Separate method for time-based exit checking independent of price updates.
         For each open position, check ONLY the time-limit condition.
         Called by a separate interval timer (set up in Plan 04).

       - handleCopySell(signal: Signal): void
         If signal.type === 'SELL' and positions.has(signal.mint):
         Close the position with reason 'copy-sell'.
         Copy-sell has priority 2 (after stop-loss, before take-profit).
         In practice this means: if stop-loss would have triggered this cycle, stop-loss wins.
         But copy-sell overrides take-profit and time-limit.

       - closePosition(mint: string, reason: ExitReason): void
         a. Get position from map
         b. Set status = 'closed'
         c. Remove from positions Map
         d. Add to history array
         e. Persist state
         f. Emit bus.emit('position:closed', position, reason)
         g. Log: "Position closed: {mint short} | Reason: {reason} | PNL: {pnlPct}%"

       - private async persist(): void
         Write positions (as array from Map values) to data/positions.json
         Write history to data/history.json
         Use writeJson from persistence utility (atomic writes)

       - getOpenPositions(): Position[] -- return Array.from(positions.values())
       - getActiveMints(): string[] -- return mints of open positions (for price monitor)

    2. Export function initPositionManager(config: Config): PositionManager
       - Create instance, call init(), return it
       - Subscribe to bus.on('trade:executed', handler) to call openPosition
       - Subscribe to bus.on('signal:detected', handler) for SELL signals to call handleCopySell
       - Subscribe to bus.on('price:updated', handler) to call checkExits

    The randomized exit params are LOCKED decisions from CONTEXT.md -- TP 100-200% (2x-3x), SL -30% to -50%, time 15-45 min. Use uniform random distribution within these ranges.
