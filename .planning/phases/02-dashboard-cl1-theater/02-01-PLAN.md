  <action>
    1. Install ws and @types/ws: `npm install ws @types/ws`

    2. Create `src/server/ws-broadcast.ts`:
       - Import WebSocketServer, WebSocket from 'ws', Server from 'http', bus from events
       - Export `initWebSocketServer(server: Server, getPositions: () => Position[]): WebSocketServer`
       - Subscribe to ALL pipeline bus events and broadcast as JSON `{ type, data, ts }`:
         - position:opened, position:updated, position:closed (include reason for closed)
         - signal:detected, signal:validated, signal:rejected (include reason for rejected)
         - trade:executed
         - price:updated (convert Map to plain object with Object.fromEntries)
       - On 'connection': send snapshot message `{ type: 'snapshot', data: { positions: getPositions() } }`
       - Add heartbeat with 30s ping/pong interval to detect dead connections
       - Throttle price:updated broadcasts to max once per 5 seconds (avoid flood)
       - Handle ws errors silently (ws.on('error', () => {}))

    3. Modify `src/server/index.ts`:
       - Add `import { createServer } from 'http'` and `import { join } from 'path'`
       - Add `import { initWebSocketServer } from './ws-broadcast.js'`
       - Add `app.use(express.static(join(process.cwd(), 'public')))` BEFORE route handlers
       - Replace `app.listen(config.port, ...)` with:
         ```
         const server = createServer(app);
         initWebSocketServer(server, () => positionManager.getOpenPositions());
         server.listen(config.port, () => { ... });
         ```
       - Keep all existing webhook/health routes unchanged

    4. Add WS message type to `src/core/types.ts`:
       ```typescript
       export interface WsMessage {
         type: string;
         data: unknown;
         ts: number;
       }
       ```
  </action>
